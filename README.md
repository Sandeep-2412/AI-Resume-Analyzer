# 🤖 AI Resume Analyzer

An **AI-powered resume analysis platform** that evaluates resumes against job descriptions and provides ATS-focused feedback to help job seekers improve their chances of getting shortlisted.

This project is built using a **serverless architecture**, eliminating the need for traditional backend infrastructure while leveraging AI for intelligent resume insights.

---

## 🚀 Repository

🔗 https://github.com/Sandeep-2412/AI-Resume-Analyzer

---

## ✨ Features

- 📄 Upload Resume (PDF only)
- 📋 Add Job Description
- 🧠 AI Resume Matching
- 📊 ATS Compatibility Score
- 📉 Detailed Feedback:
  - Tone & Language
  - Skills Match
  - Resume Content
  - Structure Optimization
- 🧩 Resume Preview
- 🔐 Secure Authentication
- ☁️ Cloud Storage
- 📂 Resume History Tracking
- 🧹 Wipe Stored Data Option

---

## 🧠 AI Analysis

The AI engine evaluates:

- Keyword Matching
- Missing Skills
- Resume Relevance
- ATS Compatibility
- Resume Strengths & Weaknesses
- Actionable Improvement Suggestions

---

## 🏗 Architecture

This project follows a **serverless model**:

- No backend server required
- AI processing handled via cloud services
- Authentication handled securely
- Resume storage managed through cloud APIs

## ⚙️ How It Works

1. User logs in
2. Uploads resume
3. Adds job description
4. Resume converted for preview
5. AI analyzes resume against job role
6. ATS score generated
7. Detailed improvement feedback displayed
